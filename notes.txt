#  3. API URL banana (Dynamic URL)

```js
const url = (city) =>
  `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${apikey}`;
```

 Ye function hai jo:

* city ka naam leta hai
* API URL banata hai

 `${city}` = dynamic value (template string)

Example:

```
Lahore → https://api.openweathermap.org/...q=Lahore
```

#  4. API se data fetch karna (IMPORTANT)

```js
async function getWeatherByLocation(city) {
  const resp = await fetch(url(city), { origin: "cors" });
  const respData = await resp.json();

  console.log(respData);

  addWeatherToPage(respData);
}
```

###  Yahan 3 important cheezein hain:

### 1. `async/await`

* async function → async ka matlab wait kar sakta hai
* await → jab tak data na aaye, ruk jao

### 2. `fetch()`

* API call karta hai (internet se data lata hai)

### 3. `resp.json()`

* API data JSON format me hota hai
* isko JS object me convert karta hai

 `console.log()` debugging ke liye use hota hai

#  5. Data ko screen par show karna

```js
function addWeatherToPage(data) {
  const temp = KtoC(data.main.temp);
```

 `data.main.temp`:

* API se temperature milta hai (Kelvin me)

---

##  Kelvin → Celsius convert

```js
function KtoC(K) {
  return Math.floor(K - 273.15);
}
```

 API temperature Kelvin me deta hai
 Tum use Celsius me convert kar rahe ho

#  6. Form submit event

```js
form.addEventListener("submit", (e) => {
  e.preventDefault();

  const city = search.value;

  if (city) {
    getWeatherByLocation(city);
  }
});
```

##  Important:

* `submit` → jab user enter press kare
* `e.preventDefault()` → page reload hone se rokta hai

 phir:

* input se city li
* function call ki
